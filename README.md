# rn-thermal-printer

Native bridge for thermal printer. Support Bluetooth, TCP, and USB printer.

## Installation

```sh
# npm
npm install rn-thermal-printer

# yarn
yarn add rn-thermal-printer
```

### Printer Bluetooth

### Printer TCP

### Printer USB

```js

```
